# CariMakan
Submission for [Menjadi Front-End Web Developer Expert](https://www.dicoding.com/academies/219) class from Dicoding

## Getting started
install dependency
```shell
npm install
```

run in development mode
```shell
npm run start-dev
```

run test
```shell
npm run test
```

run end-to-end testing
```shell
npm run e2e
```
> Note: you need to run the app first!

build the app
```shell
npm run build
```
